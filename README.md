# SoftUni ReactJS Full Course 2025

Coursework & projects for the **SoftUni ReactJS Full Course 2025** — modern React with components, hooks, Context, client-side routing, state patterns, async APIs, testing, TypeScript, Vite, and React Router.

---

## Overview

This repository tracks my progress through the SoftUni curriculum via labs, homework, mini-apps, and a capstone. The focus is on building production-ready single-page applications (SPAs): composing reusable components, managing state with hooks and the Context API, integrating APIs, adding robust routing, writing tests, and deploying with a modern toolchain.

---

## Highlights

- **Core React:** components, props, state, effects, and custom hooks
- **State management:** Context API and practical state patterns
- **Routing:** nested routes, guards, loaders/actions with React Router
- **Data fetching:** `fetch`/`async`–`await`, error & loading states, API abstraction
- **Testing:** unit & integration tests (Jest + React Testing Library)
- **Tooling:** TypeScript, Vite, ESLint, Prettier, environment configs
- **Deployment:** optimized builds and shipping to modern hosts
